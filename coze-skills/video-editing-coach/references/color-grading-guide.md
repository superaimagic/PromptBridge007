# 调色参考指南

## 一级校色（Primary Correction）——还原真实

目标：让所有镜头的曝光、色温、对比度保持一致

| 步骤 | 操作 | 工具 | 目标 |
|------|------|------|------|
| 1. 白平衡 | 色温+色调调整 | 色轮/滑块 | 确保白色是白色 |
| 2. 曝光 | 整体亮度 | 直方图 | 避免过曝/欠曝 |
| 3. 对比度 | 亮暗部差值 | 对比度滑块 | 画面通透 |
| 4. 高光/阴影 | 四段式亮度 | Lift/Gamma/Gain | 保留细节 |
| 5. 饱和度 | 全局/自然饱和度 | 饱和度滑块 | 自然不过艳 |

## 二级调色（Secondary Correction）——风格化

### HSL调整
针对特定颜色独立调整色相/饱和度/亮度
- 天空更蓝：选中蓝色 → 饱和度+20% → 亮度-10%
- 植被更绿：选中绿色 → 色相偏青 → 饱和度+15%
- 肤色保护：选中肤色范围 → 锁定，不影响其他区域

### 曲线调色
RGB曲线和色相曲线是调色的核心武器
- S曲线：增加对比度和电影感
- 暗部提亮曲线：模拟胶片褪色效果
- 色相vs色相曲线：精确改变特定颜色的色相

### 蒙版/窗口
- 圆形窗口：面部提亮
- 渐变窗口：天空压暗
- 钢笔窗口：不规则形状局部调整

## 风格化调色方案

### 电影感
- 特征：低饱和 + 青橙对比 + 颗粒感
- 参数：饱和度 -15% / 暗部偏青 / 亮部偏橙 / 颗粒 5-10%

### 日系清新
- 特征：高亮度 + 低对比 + 偏青绿 + 阴影提亮
- 参数：曝光 +0.5 / 对比度 -20% / 阴影 +15% / 色温偏冷

### 赛博朋克
- 特征：高饱和霓虹色 + 高对比 + 暗部压死
- 参数：对比度 +30% / 饱和度 +20% / 品红+青色调

### 复古胶片
- 特征：偏黄绿 + 暗部偏红 + 颗粒 + 褪色
- 参数：色温偏暖 / 阴影偏红 / 褪色 10-15% / 颗粒 10-15%

### 莫兰迪色系
- 特征：低饱和 + 灰调 + 高级感
- 参数：饱和度 -30% / 对比度 -15% / 高光压暗

## LUT使用原则

1. LUT是起点不是终点——加载后必须微调
2. 技术LUT用于LOG素材还原（S-Log3 → Rec.709）
3. 创意LUT建议不透明度60-80%
4. 同系列视频必须使用一致的LUT
5. 自制LUT：基于常用参数导出，形成个人风格
