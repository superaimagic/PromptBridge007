# 剪辑软件选择与核心操作指南

## 软件选择决策树

```
需要日更短视频、追求效率？
  ├── 是 → 剪映/CapCut专业版
  └── 否 → 需要与AE/PS联动做特效？
            ├── 是 → Premiere Pro
            └── 否 → 调色要求极高？
                  ├── 是 → DaVinci Resolve
                  └── 否 → Mac用户？
                        ├── 是 → Final Cut Pro
                        └── 否 → 剪映或PR
```

建议：至少精通一个主力软件 + 熟悉剪映（AI功能太好用了）

---

## 剪映/CapCut专业版

**适用**：短视频日常、轻量商业、团队批量
**优势**：AI功能最强、模板丰富、学习曲线最低、抖音生态打通

### 必背快捷键
| 操作 | 快捷键 |
|------|--------|
| 分割 | Ctrl+B / Cmd+B |
| 删除 | Delete |
| 撤销 | Ctrl+Z / Cmd+Z |
| 播放/暂停 | Space |
| 缩放时间线 | Ctrl+滚轮 |

### 关键操作
1. AI字幕：文本 → 智能字幕 → 自动识别 → 校对
2. 智能抠像：选中片段 → 画面 → 抠像 → 智能抠像
3. 曲线变速：选中片段 → 变速 → 曲线变速 → 自定义
4. 关键帧动画：选中素材 → 添加关键帧 → 调整属性 → 添加结束关键帧

---

## Adobe Premiere Pro

**适用**：商业项目、多平台生产、团队协作
**优势**：行业标准、AE/AU/PS联动、插件生态最丰富

### 必背快捷键
| 操作 | 快捷键 |
|------|--------|
| 波纹剪辑（Ripple） | Q（左）/ W（右） |
| 剃刀工具 | C |
| 选择工具 | V |
| 回放控制 | J（倒）/ K（停）/ L（放） |
| 入点/出点 | I / O |
| 插入/覆盖 | , / . |
| 缩放时间线 | Alt+滚轮 |

### 关键工作流
1. 代理剪辑：导入 → 代理 → 创建代理 → 选低分辨率格式
2. 多机位：选中素材 → 右键 → 创建多机位源序列
3. 动态链接AE：右键素材 → 替换为After Effects合成
4. Lumetri调色：窗口 → Lumetri颜色 → 基本校正 → 创意 → 曲线

---

## DaVinci Resolve

**适用**：高端调色、影视级项目
**优势**：免费版极强、调色行业第一、Fairlight专业音频、Fusion节点特效

### 核心概念
- 节点式调色：每个节点是一个调色步骤，可串联/并联
- 串行节点：从上到下依次处理
- 并行节点：同时处理不同区域
- 图层节点：叠加混合

### 必学操作
1. 一级校色：色轮调整Lift/Gamma/Gain
2. 二级调色：限定器选取 → HSL调整 → 窗口遮罩
3. 肤色校正：矢量示波器 → 肤色线对齐
4. 节点管理：Alt+S（添加串行节点）、Alt+P（添加并行节点）

---

## Final Cut Pro

**适用**：Mac生态、快节奏剪辑、个人高效产出
**优势**：M芯片性能炸裂、磁力时间线、买断制

### 磁力时间线逻辑
- 主故事线：核心素材
- 连接片段：B-Roll/字幕/音效附加到主故事线
- 移动主片段时连接片段自动跟随

### 必学操作
1. 磁性连接：素材自动吸附，防止轨道间隙
2. 复合片段：多素材打包为嵌套片段
3. 角色分配：对话/音乐/效果分轨管理
4. 代理工作流：导入 → 转码 → 创建代理媒体
