## 横屏 16:9

### B站
| 参数 | 推荐值 | 备注 |
|------|--------|------|
| 分辨率 | 1920×1080 | 3840×2160 (4K) |
| 帧率 | 30fps | 60fps + 120fps 获"高画质"标识 |
| 码率 | 10-15 Mbps | 40-60 Mbps (4K60) |
| 编码 | H.264 | H.265 更小体积 |
| 音频 | AAC 320kbps | |

### YouTube
| 参数 | 推荐值 |
|------|--------|
| 分辨率 | 1920×1080（最低），建议4K |
| 帧率 | 24fps（电影感）/ 30fps（常规）/ 60fps（游戏） |
| 码率 | 1080p30: 10-15 Mbps / 4K60: 40-60 Mbps |
| 编码 | H.264（兼容性最佳） |

## 通用导出检查清单

- [ ] 分辨率与源素材匹配（不升采样）
- [ ] 帧率与源素材一致（不插帧）
- [ ] 码率足够（不过度压缩）
- [ ] 音频正常（无削波、无静音、声道正确）
- [ ] 色彩空间正确（Rec.709）
- [ ] 无黑帧、无花屏
- [ ] 字幕已校对（无错别字、断句正确）
- [ ] 全片从头到尾完整播放一遍
