# 私域流量运营

## 任务目标
- 搭建私域流量运营体系
- 设计用户分层和精准触达策略
- 优化私域转化路径和GMV增长
- 管理微信个人号/企业微信/社群矩阵

## 执行流程
1. **资产盘点**：梳理现有私域触点（公众号/个人号/企业微信/社群/小程序）
2. **体系搭建**：设计引流→留存→转化→裂变的完整闭环
3. **用户分层**：基于RFM模型进行用户分层和差异化运营
4. **内容策略**：朋友圈/SOP/社群内容规划
5. **数据优化**：追踪LTV、转化率、复购率

## 输入
- `business_type`: 业务类型（必填）
- `target_users`: 目标用户画像（必填）
- `current_channels`: 现有私域触点（可选）
- `product_info`: 产品/服务信息（必填）

## 输出
- 私域运营方案：触点矩阵 + 引流策略 + 转化路径
- 用户分层模型：RFM分析 + 差异化运营策略
- 内容SOP：朋友圈/社群/1v1触达的内容模板
- 数据看板：核心指标定义 + 追踪方案

## 注意事项
- 微信平台规则严格，避免诱导分享和批量加好友
- 私域的本质是关系和信任，不是群发广告
- 用户隐私保护是底线，不滥用用户数据
- 前3个月重在养信任，别急着卖货
