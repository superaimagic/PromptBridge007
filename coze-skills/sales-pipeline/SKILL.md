# 销售管道管理

## 任务目标
- 分析销售管道健康状况
- 优化销售漏斗各阶段转化率
- 进行销售预测和风险评估
- 制定CRM策略和销售流程改进方案

## 执行流程
1. **管道审计**：分析各阶段商机数量、金额、转化率、周期
2. **瓶颈识别**：找出转化率最低的阶段，分析原因
3. **策略制定**：设计针对性改进措施和SOP
4. **预测建模**：基于历史数据建立销售预测模型
5. **持续优化**：追踪关键指标，迭代策略

## 输入
- `pipeline_data`: 销售管道数据（必填）
- `sales_cycle`: 销售周期（必填）
- `target_revenue`: 目标收入（可选）
- `team_size`: 团队规模（可选）

## 输出
- 管道健康报告：各阶段商机分布 + 转化率 + 平均周期
- 瓶颈分析：问题诊断 + 根因分析
- 改进方案：SOP优化 + 话术模板 + 跟进节奏
- 销售预测：季度/年度预测 + 置信区间

## 注意事项
- 管道数据需定期更新，过期数据影响分析准确性
- 不同行业销售周期差异大，需针对性分析
- 预测模型仅供参考，市场变化会影响准确性
