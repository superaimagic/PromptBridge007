# 微信公众号全案运营

## 任务目标
- 制定公众号内容策略和推送节奏
- 打造高打开率的图文内容（目标打开率 > 30%）
- 搭建和运营微信社群体系
- 设计裂变增长模型和转化路径
- 构建完整的私域流量闭环

## 执行流程
1. **现状诊断**：分析公众号数据（关注量/打开率/阅读完成率/取关率）、评估社群状态
2. **策略制定**：明确内容定位、设计内容日历、规划社群体系和裂变路径
3. **执行落地**：产出公众号内容、搭建和运营社群、执行裂变活动
4. **数据驱动优化**：追踪核心指标、AB测试、持续迭代

## 输入
- `brand_info`: 品牌/业务信息（必填）
- `target_audience`: 目标用户画像（必填）
- `current_status`: 当前公众号/私域现状（可选）
- `business_goal`: 商业目标（品牌认知/线索获取/销售转化/用户留存，可选）

## 输出
- 内容策略文档：用户画像 + 内容支柱 + 编辑日历 + 菜单架构
- 公众号爆文模板：标题二选一 + 摘要 + 封面图 + 正文结构
- 社群运营SOP：每日节奏 + 每周活动 + 角色分工 + 关键指标
- 裂变活动方案：模型设计 + 海报要素 + 路径优化 + 风控

## 注意事项
- 严格遵守微信平台规则，不诱导分享、不诱导关注
- 推送频率克制：宁可少发也不要让用户取关
- 社群不刷屏、不频繁发广告，价值内容 > 促销信息
- 内容配比：60% 价值内容 + 30% 互动内容 + 10% 推广内容
- 私域的本质是"用户愿意持续看你的内容"
