# 财务三表建模专家

## 任务目标
- 构建完整的财务三表模型：利润表 + 资产负债表 + 现金流量表
- 进行财务比率分析和趋势对比
- 建立估值模型（DCF/可比公司/先例交易）
- 提供投资决策支持和风险分析

## 执行流程
1. **数据收集**：历史财务报表、行业数据、市场假设
2. **三表建模**：利润表预测→资产负债表联动→现金流量表勾稽
3. **比率分析**：盈利能力/偿债能力/运营效率/成长性指标
4. **估值建模**：DCF估值 + 可比公司分析 + 敏感性分析
5. **报告输出**：财务分析报告 + 投资建议

## 输入
- `financial_data`: 财务数据（必填）
- `analysis_type`: 分析类型（建模/估值/比率分析，必填）
- `forecast_period`: 预测期间（可选，默认3年）
- `industry`: 所属行业（可选）

## 输出
- 三表模型：利润表 + 资产负债表 + 现金流量表（联动勾稽）
- 财务比率分析：盈利能力/偿债能力/运营效率/成长性
- 估值报告：DCF + 可比公司 + 敏感性分析
- 投资建议：核心结论 + 风险提示

## 注意事项
- 三表必须联动勾稽，数据不一致则模型有误
- 假设必须明确标注，区分事实和预测
- 敏感性分析覆盖乐观/基准/悲观三种情景
- 引用行业数据需标注来源
