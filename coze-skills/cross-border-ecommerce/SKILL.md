# 跨境电商全链路运营

## 任务目标
- 为中国卖家提供跨境电商全平台运营策略
- 覆盖选品分析、Listing优化、广告投放、物流合规、品牌出海全链路
- 制定多站点运营方案和利润测算模型
- 确保运营合规，避免账号风险

## 执行流程
1. **市场调研与选品**：利用工具分析目标市场品类数据，评估容量、竞争、利润和合规要求
2. **合规准备与账号搭建**：完成产品认证、VAT注册、商标品牌备案、店铺搭建
3. **Listing上线与优化**：多语言Listing撰写、A+页面制作、关键词布局
4. **广告投放与流量获取**：PPC广告架构搭建、平台活动报名、站外引流布局
5. **数据复盘与运营迭代**：日报/周报数据追踪、核心指标监控、策略调整

## 输入
- `target_market`: 目标市场（北美/欧洲/日本/东南亚，必填）
- `product_category`: 产品品类（必填）
- `platforms`: 目标平台（Amazon/Shopee/Lazada/Temu/TikTok Shop等，必填）
- `budget`: 启动预算范围（可选）

## 输出
- 跨境选品评估表：市场维度 + 利润维度 + 合规维度
- 多站点运营对比表：各平台核心逻辑、流量获取、物流方案、利润率
- Amazon广告投放框架：新品期/成长期/成熟期分阶段策略
- 全链路成本核算模型：采购+头程+仓储+佣金+广告+尾程+退货+汇率

## 注意事项
- 产品合规是底线：无认证的产品绝不上架
- VAT/Sales Tax必须合规申报
- 知识产权零容忍：不侵权、不跟卖
- 广告ACOS有底线：超过毛利率的广告计划必须优化
- 库存周转率纳入核心考核，FBA长期仓储费是隐形利润杀手
