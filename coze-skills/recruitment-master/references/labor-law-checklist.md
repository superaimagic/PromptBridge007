# 中国劳动法合规检查清单

## 一、劳动合同

| 检查项 | 合规标准 | 风险等级 |
|--------|----------|----------|
| 合同签订时限 | 入职30天内必须签订书面合同 | 🔴 超期需支付双倍工资 |
| 合同类型 | 固定期限/无固定期限/以完成一定任务为期限 | 🟡 连续两次固定期限后员工有权要求无固定期限 |
| 合同内容 | 必须包含九项必备条款（合同期限、工作内容、工作地点、工作时间、休息休假、劳动报酬、社会保险、劳动保护、其他） | 🔴 缺失可能导致合同无效 |
| 试用期约定 | 合同3月-1年：≤1个月；1年-3年：≤2个月；3年以上/无固定：≤6个月 | 🔴 超期需按转正工资补差 |

## 二、试用期

| 检查项 | 合规标准 | 风险等级 |
|--------|----------|----------|
| 试用期次数 | 同一用人单位与同一劳动者只能约定一次试用期 | 🔴 二次约定无效 |
| 试用期工资 | ≥约定工资的80%，且≥当地最低工资 | 🔴 不足需补差 |
| 试用期社保 | 试用期内必须缴纳社保和公积金 | 🔴 未缴属违法 |

## 三、五险一金

| 险种 | 单位比例 | 个人比例 | 备注 |
|------|----------|----------|------|
| 养老保险 | 16% | 8% | 各地比例有差异 |
| 医疗保险 | 8-10% | 2% | |
| 失业保险 | 0.5-1% | 0.5% | |
| 工伤保险 | 0.2-1.9% | 0% | 行业差别费率 |
| 生育保险 | 0.5-1% | 0% | 已并入医疗保险 |
| 住房公积金 | 5-12% | 5-12% | 单位和个人同比例 |

## 四、竞业限制

| 检查项 | 合规标准 | 风险等级 |
|--------|----------|----------|
| 限制期限 | ≤2年 | 🔴 超期无效 |
| 补偿金 | ≥劳动合同终止前12个月平均工资的30%，且≥当地最低工资 | 🔴 3个月未付补偿员工可解除 |
| 适用范围 | 高管、高级技术人员、其他负有保密义务的人员 | 🟡 不应扩大到全员 |

## 五、离职与补偿

| 情形 | 补偿标准 | 说明 |
|------|----------|------|
| 协商一致解除 | N | 双方协商 |
| 无过失性辞退（未提前30天通知） | N+1 | 代通知金 |
| 违法解除 | 2N | 赔偿金 |
| 经济性裁员（≥20人或≥10%） | N | 需提前30天通知工会+报劳动部门 |
| 员工主动辞职 | 0 | |
| N的计算 | 每满1年=1个月；6月-1年=1个月；不满6月=0.5个月 | 月工资上限为当地社平工资3倍，最高12年 |

## 六、禁止就业歧视

JD和招聘过程中不得包含以下歧视性要求：
- ❌ 性别限制（"只招男性"）
- ❌ 年龄限制（"35岁以下"）
- ❌ 婚育状况限制（"已婚已育优先"）
- ❌ 民族/宗教限制
- ❌ 户籍限制（部分岗位除外）
- ❌ 外貌/身高限制（与岗位无关的）

## 七、个人信息保护（PIPL）

- 收集候选人个人信息前必须获得明确授权
- 背景调查需要候选人预先书面同意
- 未录用候选人的个人信息应在合理期限内删除
- 不得向第三方泄露候选人信息
