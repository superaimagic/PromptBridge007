# 结构化面试题库

## 按岗位级别分类

### 初级岗位（0-3年经验）

#### 通用能力
1. 请描述你在学校/上一份工作中完成的一个最满意的项目。你承担了什么角色？遇到了什么困难？如何解决的？
2. 当你同时收到多个任务时，你如何安排优先级？请举一个具体例子。
3. 描述一次你与团队成员产生分歧的经历。你是如何处理的？

#### 学习能力
4. 你最近自学的一项新技能是什么？花了多长时间？学习路径是怎样的？
5. 如果入职后遇到一个完全不懂的技术/业务问题，你的第一步会做什么？

#### 职业动机
6. 你为什么选择我们公司？你对这个岗位的理解是什么？
7. 你未来3年的职业规划是什么？

---

### 中级岗位（3-7年经验）

#### 专业能力
1. 请描述一个你主导的最复杂的技术/业务项目。从需求到交付的全流程是怎样的？
2. 你在这个项目中做的最关键的决策是什么？如果重来一次，你会改变什么？
3. 请分享一个你通过技术/流程优化带来显著效率提升的案例。

#### 问题解决（STAR法）
4. Situation：描述一个你遇到的最棘手的工作问题
5. Task：你的任务/目标是什么
6. Action：你采取了哪些具体行动
7. Result：最终结果如何（请量化）

#### 团队协作
8. 你如何与产品经理/其他部门协作？遇到过什么冲突？如何化解？
9. 如果你发现团队的技术方案有明显问题，但TL坚持原方案，你会怎么做？

---

### 高级/管理岗位（7年+经验）

#### 领导力
1. 描述你带过的最大规模的团队。你如何分工、激励和考核？
2. 分享一个你招聘过的最好的人和一个招错的人。你从中学到了什么？
3. 团队中出现"老油条"员工，你怎么处理？

#### 战略思维
4. 如果让你从零搭建我们公司的[XX体系]，你的规划思路是什么？
5. 你认为[行业]未来3-5年的发展趋势是什么？我们应该如何提前布局？

#### 危机处理
6. 描述一个你经历过的重大线上事故/业务危机。你当时的决策过程是怎样的？
7. 如果公司要求你在3个月内将团队产出翻倍，你会怎么做？

## 按面试类型分类

### 行为面试（STAR法）

| 考察维度 | 问题示例 |
|----------|----------|
| 主动性 | "请举一个你主动承担额外责任的例子" |
| 抗压能力 | "描述一次你在高压环境下完成工作的经历" |
| 冲突处理 | "你和同事/上级有过意见分歧吗？你是如何处理的？" |
| 失败反思 | "请分享一次工作上的重大失误。你从中学到了什么？" |
| 创新思维 | "你提出过哪些被采纳的改进建议？带来了什么效果？" |

### 技术面试

| 考察维度 | 评估要点 |
|----------|----------|
| 基础知识 | 深度而非广度，追问原理而非记忆 |
| 问题解决 | 给一个实际问题，观察解题思路和沟通方式 |
| 系统设计 | 可扩展性、高可用、数据一致性等维度 |
| 代码质量 | 可读性、可维护性、边界处理、测试意识 |

### 无领导小组讨论

| 观察维度 | 评分标准 |
|----------|----------|
| 角色承担 | 领导者/协调者/计时者/记录者/贡献者 |
| 讨论贡献 | 观点质量而非数量 |
| 团队协作 | 是否倾听他人、是否推动共识 |
| 逻辑表达 | 条理清晰、论据充分 |

## 面试评分表模板

| 评估维度 | 权重 | 评分(1-5) | 评分理由 |
|----------|------|-----------|----------|
| 专业能力 | 40% | | |
| 问题解决 | 20% | | |
| 沟通表达 | 15% | | |
| 团队协作 | 15% | | |
| 文化匹配 | 10% | | |
| **综合评分** | 100% | | |

评分标准：
- 5分：超出预期，在同级别中表现卓越
- 4分：符合预期，能够胜任岗位要求
- 3分：基本符合，部分维度有待提升
- 2分：低于预期，存在明显差距
- 1分：严重不符合，不予考虑
