# 标准化入职检查清单

## 入职前（T-7天）
- [ ] 发送入职通知邮件/短信，附材料清单
- [ ] 准备工作站、电脑、门禁卡等办公资源
- [ ] 开通企业邮箱、OA系统、飞书/钉钉/企微账号
- [ ] 通知用人部门和指定导师准备迎新
- [ ] 安排入职培训日程

## 入职当天（T日）
- [ ] 签订劳动合同、保密协议、员工手册确认书
- [ ] 完成社保和公积金登记
- [ ] 录入HRIS系统（北森/i人事/飞书People等）
- [ ] 发放员工手册和IT使用指南
- [ ] 入职培训：公司文化、组织架构、制度流程
- [ ] 用人部门迎新和团队介绍
- [ ] 与指定导师进行第一次1对1沟通

## 第一周（T+1至T+7天）
- [ ] 确认岗位职责和试用期目标
- [ ] 安排业务培训和系统操作培训
- [ ] HR进行入职体验回访
- [ ] 将新人加入部门沟通群和相关项目组

## 第一个月（T+30天）
- [ ] 导师进行首月反馈面谈
- [ ] HR进行新人满意度调查
- [ ] 确认试用期考核计划和里程碑目标

## 试用期管理
- [ ] 明确试用期考核标准和评估时间点（建议每月或双月评估）
- [ ] 建立试用期预警机制：对表现不佳的新人主动沟通改进计划
- [ ] 试用期不通过的处理流程：充分记录、合法合规终止、尊重沟通
