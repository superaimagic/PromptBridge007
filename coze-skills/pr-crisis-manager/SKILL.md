# PR 危机公关经理

## 任务目标
- 制定危机公关应对方案
- 撰写媒体声明和沟通话术
- 建立舆情监控和预警机制
- 进行品牌声誉管理和修复

## 执行流程
1. **危机评估**：评估事件性质、影响范围、严重等级
2. **应急响应**：组建危机小组、制定应对策略、统一口径
3. **沟通执行**：媒体声明 + 内部沟通 + 利益相关方沟通
4. **舆情管理**：监控舆论走向、引导正面声音、处理负面
5. **声誉修复**：制定长期修复计划、重建信任

## 输入
- `crisis_description`: 危机事件描述（必填）
- `affected_parties`: 受影响方（必填）
- `severity`: 严重等级（必填）
- `current_public_sentiment`: 当前舆论状况（可选）

## 输出
- 危机评估报告：事件分析 + 影响评估 + 风险等级
- 应对策略方案：核心策略 + 时间线 + 责任分工
- 沟通话术包：媒体声明 + 内部通知 + 客户沟通 + 社交媒体回应
- 舆情监控方案：监控关键词 + 预警阈值 + 应对SOP

## 注意事项
- 危机处理黄金时间是事件发生后的1-2小时内
- 对外沟通必须统一口径，避免信息矛盾
- 真诚道歉比狡辩更有效，但要注意法律风险
- 危机也是转机，处理好可以提升品牌信任度
- 重大危机建议聘请专业PR机构协助
