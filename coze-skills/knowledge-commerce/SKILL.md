# 知识付费运营策略

## 任务目标
- 设计知识付费产品体系（课程/训练营/社群/咨询）
- 制定定价策略和促销方案
- 优化转化漏斗和用户生命周期价值
- 分析用户学习行为和留存

## 执行流程
1. **市场调研**：分析赛道竞争、用户需求、价格带
2. **产品设计**：课程大纲 + 交付形式 + 服务体系
3. **定价策略**：阶梯定价 + 促销节奏 + 组合套餐
4. **转化漏斗**：引流→体验→信任→成交→复购
5. **用户运营**：学习激励 + 社群互动 + 口碑裂变

## 输入
- `knowledge_domain`: 知识领域（必填）
- `target_audience`: 目标学员画像（必填）
- `product_type`: 产品类型（课程/训练营/社群/咨询，必填）
- `team_capability`: 团队能力（可选）

## 输出
- 产品设计方案：课程大纲 + 交付节奏 + 服务体系
- 定价方案：价格锚点 + 阶梯策略 + 促销日历
- 转化漏斗方案：各环节策略和转化率目标
- 运营SOP：日常运营 + 活动策划 + 口碑裂变

## 注意事项
- 知识付费的核心是交付效果，不是课程数量
- 定价需考虑用户付费能力和竞品价格带
- 完课率和口碑是长期增长的基础
- 不夸大课程效果，保持交付承诺与实际一致
