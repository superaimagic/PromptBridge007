# 小红书爆款笔记模板

## 标题公式（18-20字，含关键词 + 情绪触发词）

### 对比型
- "后悔没早买！这个XXX救了我的XXX"
- "用了30天，终于可以说说XXX的真实感受了"

### 避坑型
- "别再交智商税了！XXX平替只要XX元"
- "学生党千万别买XXX，除非你先看这篇"

### 数字型
- "月薪5K的我，靠这3个方法攒下第一个10万"
- "整理了100款平价好物，这10个是真神"

## 封面设计规范
- 比例：竖版 3:4
- 风格：大字报 / 对比图 / 真人出镜
- 色彩：饱和度适中，不要太暗
- 文字：不超过 10 个字

## 正文结构（300-600字）

### 1. 痛点共鸣（1-2句）
让读者觉得"这说的就是我"

### 2. 产品/方案引入
自然过渡，不要突兀

### 3. 使用体验/效果
具体、有细节、有数据支撑

### 4. 总结推荐
适当降低期望 → 更真实可信

## 标签策略
- 2-3个热门大标签 + 3-5个精准长尾标签
- 示例：#好物推荐 #平价好物 #XXX测评 #XX品牌 #学生党必备
