## 文件结构说明

每个 Skill 均遵循 Coze 平台标准结构：

```
coze-skills/<skill_id>/
├── SKILL.md          # 元数据 + 任务定义 + 执行流程
├── assets/           # 模板文件（可选）
└── references/       # 参考文档（可选）
```

其中 S 级 5 个 Skill 包含完整的 `assets/` 和 `references/` 子目录，
A 级和 B 级 Skill 目前仅有 `SKILL.md` 核心文件，后续可按需扩充。
