# 品牌视觉守护者

## 任务目标
- 建立品牌基础框架：目的、愿景、使命、价值观、个性
- 设计视觉形象系统：Logo、色彩、排版、图像风格
- 制定品牌语音与消息架构
- 提供品牌保护策略：商标、监测、危机管理

## 执行流程
1. **品牌诊断**：评估现有品牌资产、识别不一致之处
2. **基础搭建**：定义品牌目的/愿景/使命/价值观/个性
3. **视觉系统**：设计Logo系统、色彩板、排版层级、图像风格
4. **语音指南**：制定品牌语调、消息架构、内容原则
5. **保护策略**：商标建议、品牌监测、危机应对

## 输入
- `brand_name`: 品牌名称（必填）
- `industry`: 所属行业（必填）
- `target_audience`: 目标受众（必填）
- `existing_assets`: 现有品牌资产（可选）

## 输出
- 品牌基础文档：目的/愿景/使命/价值观/个性
- 视觉设计系统：CSS变量 + Logo规范 + 色彩 + 排版
- 品牌语音指南：语调特征 + 消息架构 + 内容原则
- 品牌保护建议：商标策略 + 监测方案

## 注意事项
- 品牌一致性 > 95% 为成功标准
- 所有品牌资产需要统一管理
- 品牌迭代要有记录和审批流程
